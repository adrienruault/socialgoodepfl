Propose your project

Is your project eligible?

Your project must respect three conditions in order to be published on socialgoodepfl:

  • It has to be produced by epfl students or staff
  • It must be open-source
  • It has to somehow benefit society in a disinterested manner

What we need to publish it

If your project is eligible and you are interested in publishing it on socialgoodepfl please contact us with the material indicated below:

  • A Title describing as explicitely as possible your project in no more than 140 characters
  • A Picture in jpeg format of maximum 150x150 pixels to illustrate your project (eg. an screenshot or a logo).
  • A Description that conveys the idea of your project. It can have the length you want, you are totally free on that point.
  • Any External resources that can be used to access your project or its results (eg. a github or a blog post link)
  • The Contact informations of the team members or the organization that is undertaking the project (eg. email, linkedin, website, twitter, github, etc...)

How your project will be displayed

All the projects published on socialgoodepfl are associated to their own descriptive page. This page contains the project title, a picture to illustrate it, a description along with links to external resources and finally the contact informations of the project authors.

The descriptive page of each project is accessible in two ways. Firstly each project is referenced in the exhaustive project list of socialgoodepfl. Moreover every project is referenced in the home page's gallery thanks to a card that contains the title of the project and a picture to illustrate it.
